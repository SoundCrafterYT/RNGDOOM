RNGDOOM by SoundCrafter
Version 0.97 OBT

Get ready for some randomized action!

RNGDOOM is a Doom 2 mod that randomizes mostly weapons, sounds, monster AI, items, etc

[Requirements]
Windows 7 Basic 32-bit (minimum), Windows 10 Pro x64
NVIDIA Geforce RTX 2060 or better
AMD Ryzen 5 2600x 6-Core or better
8GB RAM or higher

[Compatibility]
This mod is compatible with GZDOOM 4.11 and up
